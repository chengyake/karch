
-N 告诉SSH客户端，这个连接不需要执行任何命令。仅仅做端口转发
-f 告诉SSH客户端在后台运行
-L 做本地映射端口，-L X:Y:Z的含义是，将IP为Y的机器的Z端口通过中间服务器映射到本地机器的X端口。
-R 做远程映射端口, -R X:Y:Z的含义是，就是把我们内部的Y机器的Z端口映射到远程机器的X端口上。 



1. 把内网的22端口映射到远程123.xxx的2222上
在内网机器上执行，
ssh -N -f -R 2222:127.0.0.1:22 username@123.123.123.123
在外网的IP上登陆
ssh -p 2222 username@localhost

后台保持连接:
ssh -f -R 2222:localhost:22 123.123.123.123 "vmstat 30"
